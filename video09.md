# Personalización de Escritorio Windows 🚀

Este repositorio contiene las herramientas, configuraciones y recursos utilizados para personalizar el entorno de escritorio de Windows, logrando una estética limpia y funcional.

## 🛠️ Herramientas Utilizadas

Según las pestañas activas en la demostración, se utilizaron los siguientes componentes para la personalización:

1. **Winstep Nexus Dock:** El sistema de acoplamiento avanzado para Windows que añade una barra de aplicaciones estilo macOS.
2. **Rainmeter:** Software de personalización de escritorio para mostrar skins personalizables (como el skin *Robik*).
3. **Windhawk:** Herramienta para modificar y tunear el comportamiento y diseño del sistema operativo Windows.
4. **Gits Cursors v3:** Pack de cursores personalizados para cambiar el puntero del mouse predeterminado.

---

## 🚀 Cómo Replicar esta Configuración

1. **Descarga los programas principales:** Visita las páginas oficiales de Winstep Nexus, Rainmeter y Windhawk.
2. **Instala los temas:** Descarga los skins de Rainmeter y muévelos a tu carpeta de documentos de Rainmeter.
3. **Configura el Dock:** Ejecuta Nexus Dock y personaliza los íconos a tu gusto.
